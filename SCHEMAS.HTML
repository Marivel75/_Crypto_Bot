stateDiagram-v2
    [*] --> RuleEvaluation: Indicateurs prets
    RuleEvaluation --> Aggregation: Regles declenchees
    RuleEvaluation --> Supprime: Aucune regle
    Aggregation --> BlendML: ML disponible
    Aggregation --> CheckConfidence: Rules seules
    BlendML --> CheckConfidence
    CheckConfidence --> Supprime: Confidence inferieure a 0.6
    CheckConfidence --> CheckFees: Confidence suffisante

    Evalue --> Correct: Direction confirmee
    Evalue --> Incorrect: Direction inversee

    API-->>UI: ApiResponse avec data

    API->>LLM: System prompt + message

    API-->>UI: Reply + disclaimer

Authentification JWT

Flux complet : connexion bcrypt, generation JWT HS256, et acces aux ressources protegees.
Scroll = zoom | Drag = pan | Double-clic = plein ecran 
sequenceDiagram
    participant U as Utilisateur
    participant F as Streamlit
    participant A as FastAPI
    participant DB as TimescaleDB
    U->>F: Email + mot de passe
    F->>A: POST /auth/login
    A->>DB: SELECT user WHERE email = ?
    DB-->>A: UserOrm
    A->>A: bcrypt verify password
    A->>A: JWT encode user_id + exp
    A-->>F: access_token
    F->>F: Stocker dans session_state
    Note over U,DB: Requetes protegees
    U->>F: Voir Portfolio
    F->>A: GET /portfolio + Bearer token
    A->>A: JWT decode + validate
    A->>DB: SELECT portfolio WHERE user_id
    DB-->>A: Entries
    A-->>F: ApiResponse portfolio
    F-->>U: Tableau des positions

Schema Base de Donnees

Les 9 tables TimescaleDB : crypto_prices (hypertable), indicateurs, signaux, news, users, portfolio, watchlist.
Scroll = zoom | Drag = pan | Double-clic = plein ecran 
erDiagram
    USERS ||--o{ PORTFOLIO : possede
    USERS ||--o{ WATCHLIST : surveille
    TRADING_SIGNALS ||--o| SIGNAL_OUTCOMES : evalue
    NEWS_ARTICLES ||--o| TEXT_MINING : analyse
    USERS {
        uuid id PK
        string username UK
        string email UK
        string password_hash
        string persona_type
        json preferences
        datetime created_at
    }
    CRYPTO_PRICES {
        string symbol PK
        string timeframe PK
        datetime timestamp PK
        decimal price_open
        decimal price_high
        decimal price_low
        decimal price_close
        decimal volume_24h
        string source
    }
    INDICATORS {
        uuid id PK
        string symbol
        string timeframe
        datetime timestamp
        decimal rsi
        decimal bollinger_upper
        decimal bollinger_lower
        string harmonic_pattern
        decimal trend_slope
        string trend_type
    }
    TRADING_SIGNALS {
        uuid id PK
        string symbol
        string signal_type
        decimal confidence_score
        string timeframe_primary
        json rules_triggered
        int leverage_suggested
        string model_version
        datetime created_at
    }
    SIGNAL_OUTCOMES {
        uuid id PK
        uuid signal_id FK
        decimal price_at_signal
        decimal price_after_1h
        decimal price_after_4h
        decimal pnl_simulated
        boolean was_correct
    }
    NEWS_ARTICLES {
        uuid id PK
        string title
        string source
        string url UK
        decimal sentiment_score
        json keywords
        datetime published_at
    }
    TEXT_MINING {
        uuid id PK
        uuid article_id FK
        json word_cloud
        text summary
        json entities
    }
    PORTFOLIO {
        uuid id PK
        uuid user_id FK
        string symbol
        decimal quantity
        decimal entry_price
        text notes
    }
    WATCHLIST {
        uuid id PK
        uuid user_id FK
        string symbol
        datetime added_at
    }
