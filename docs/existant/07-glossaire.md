# Glossaire

> **Referentiel commun pour toutes les equipes.**

| Terme | Definition |
|-------|-----------|
| **APScheduler** | Advanced Python Scheduler — librairie pour executer des jobs periodiques en Python |
| **Backtesting** | Evaluation d'une strategie de trading sur des donnees historiques |
| **Bollinger Bands** | Indicateur technique : moyenne mobile +/- 2 ecarts-types, mesure la volatilite |
| **CCXT** | CryptoCurrency eXchange Trading — librairie Python unifiee pour connecter les APIs d'exchanges |
| **DEX** | Decentralized Exchange — plateforme d'echange decentralisee sans intermediaire |
| **DVC** | Data Version Control — outil de versioning pour datasets et modeles ML |
| **ETL** | Extract, Transform, Load — pipeline de traitement de donnees |
| **Fear & Greed Index** | Indice de sentiment crypto de 0 (peur extreme) a 100 (avidite extreme) |
| **Funding rate** | Frais periodiques (8h) sur les perpetuels pour maintenir le prix proche du spot |
| **Harmonic Pattern** | Patterns graphiques bases sur les ratios de Fibonacci (Gartley, Butterfly, etc.) |
| **Hypertable** | Table TimescaleDB optimisee pour les series temporelles avec partitionnement auto |
| **Levier** | Multiplicateur de position (ex: levier 10x = controler 10x sa mise) |
| **Liquidation** | Fermeture forcee d'une position a effet de levier quand la marge est insuffisante |
| **MinIO** | Stockage objet S3-compatible self-hosted, pour datasets et modeles |
| **MLflow** | Plateforme open source pour le cycle de vie du ML (tracking, registry, deploiement) |
| **Multi-timeframe** | Analyse d'un indicateur sur plusieurs unites de temps simultanees (1h, 4h, 1D, etc.) |
| **NLP** | Natural Language Processing — traitement du langage naturel |
| **OHLCV** | Open, High, Low, Close, Volume — donnees de prix standard par bougie |
| **Perpetuels** | Contrats a terme sans date d'expiration, produits derives crypto |
| **RSI** | Relative Strength Index — indicateur de surachat (>70) / survente (<30) |
| **Sharpe ratio** | Rendement ajuste au risque : (rendement - taux sans risque) / volatilite |
| **Slippage** | Difference entre le prix attendu et le prix reel d'execution |
| **Stablecoin** | Cryptomonnaie indexee sur une monnaie fiat (ex: USDT sur le dollar) |
| **TimescaleDB** | Extension PostgreSQL optimisee pour les series temporelles |
| **Timeframe (TF)** | Unite de temps pour une bougie : 1m, 5m, 1h, 2h, 3h, 4h, 1D, 1W, 1M |
| **Trend line** | Droite sur un graphique reliant des plus bas ou plus hauts successifs |
| **Walk-forward** | Methode de backtesting glissante simulant la prise de decision en temps reel |
