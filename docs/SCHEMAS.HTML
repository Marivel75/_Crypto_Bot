RL Agent - Deep Q-Learning

A Developper
Q-Learning adapte exit strategies ; apprentissage par renforcement sur PnL realise.
Scroll = zoom | Drag = pan | Double-clic = plein ecran 
stateDiagram-v2
    [*] --> Observe
    Observe --> State: Current TF Candle
    State --> ChooseAction: Epsilon-Greedy
    ChooseAction --> Hold: Q=0
    ChooseAction --> Buy: Q>theta
    ChooseAction --> Sell: Q<-theta
    Hold --> Transition: Next Candle
    Buy --> Transition
    Sell --> Transition
    Transition --> ComputeReward: Delta PnL
    ComputeReward --> UpdateQ: Bellman Equation
    UpdateQ --> Done: Episode End?
    Done -->|No| Observe
    Done -->|Yes| [*]
Pourquoi: RL apprend adaptive exit strategies au-dela SL/TP fixes ; Q-Learning evite Monte Carlo / tabular Q complexity.
Decision: DQN avec replay buffer (10k transitions); reward = realized PnL - slippage; train sur 2-year rolling windows; inference live tous les 4h.
